The Library Management System provides a centralized platform for managing library operations such as book inventory, student records, loan tracking, and subscription management. It includes an admin dashboard with authentication and profile management features.

This project demonstrates practical usage of PHP backend logic, MySQL database operations, and basic frontend integration using HTML, CSS, and Bootstrap.
